üé® Customization Guide

Change Colors

Edit css/style.css - Look for CSS variables at the top:

  • --primary-color: #d4516b; - Main pink color
  • --primary-dark: #b83d56; - Darker shade
  • --primary-light: #ff8fa3; - Lighter shade

Add More Cakes

  1. Add cake image to assets/ folder
  2. Edit cakes.html
  3. Copy an existing cake card and modify:
    • Image source
    • Cake name
    • Description
    • Price

Update Contact Information

Search and replace in all HTML files:

  • Phone: +92 316 0608032
  • Address: DHA Phase 1, Karachi
  • Email: Add your email in contact form
